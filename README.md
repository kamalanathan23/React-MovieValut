# React Movies App

A modern movie browsing application built with React that allows users to search and explore movies using the TMDB API. Features a sleek, responsive design with trending movies tracking powered by Appwrite.


## Screenshots


![App Screenshot](https://raw.githubusercontent.com/kamalanathan23/React-MovieValut/refs/heads/main/public/movie.png)


## Tech Stack

**Client:** React, Vite, TailwindCSS

**Backend:** Appwrite, TMDB API
